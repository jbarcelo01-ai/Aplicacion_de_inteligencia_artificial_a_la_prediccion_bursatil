# Stock Market Prediction using AI

Proyecto final de la Especialización/Maestría en Inteligencia Artificial (FIUBA).

## Objetivo
Desarrollar un sistema basado en técnicas de machine learning capaz de:
- analizar información financiera,
- generar predicciones de activos,
- producir señales de inversión,
- realizar backtesting,
- y asistir la construcción de carteras.

## Tecnologías
- Python
- Pandas
- Scikit-learn
- TensorFlow / PyTorch
- APIs financieras
